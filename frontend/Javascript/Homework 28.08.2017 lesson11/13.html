Сделать сортировку таблицы при клике на заголовок. 1. Обработчик onclick можно повесить один, на всю таблицу или THEAD. Он будет игнорировать клики не на TH. 2. При клике на TH обработчик будет получать номер из TH, на котором кликнули (TH.cellIndex) и вызывать функцию sortColumn, передавая ей номер колонки. 3. Функция sortColumn(colNum) будет сортировать. 4. У вас должен быть массив студентов. Этот массив вы должны отобразить в таблице. При сортировке вы сортируете массив и перерисовываете таблицу.
